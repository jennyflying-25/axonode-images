# Axonode Images

This repository stores public image assets for Axonode's optical transceiver business, including product photos and application scenarios.

## Purpose

- Host lightweight images for Axonode independent sites, blogs, and documentation.
- Provide stable URLs for use in frontend pages and marketing materials.
- Keep source images versioned and organized in one place.

## Usage

- Images are stored under the root directory for now, e.g.:

  - `axonode_10g_lr_front.png` – 10G LR optical transceiver front view  
  - `axonode_10g_lr.png` – 10G LR optical transceiver product image  
  - `axonode_400G_ER.jpg` – 400G ER optical transceiver product image  

- To use an image in a website or document, reference the raw or CDN URL, for example:

  ```text
  https://raw.githubusercontent.com/<your-username>/axonode-images/main/axonode_10g_lr_front.png
  ```

  or via a third‑party CDN such as jsDelivr:

  ```text
  https://cdn.jsdelivr.net/gh/<your-username>/axonode-images@main/axonode_10g_lr_front.png
  ```

## Notes

- Only upload images that are safe to be **publicly** accessible.
- Large source files, customer‑specific images, and internal documents should be stored in private storage (e.g. Google Drive, NAS) instead.
- Please keep file names descriptive (brand, model, spec, view) for easier searching.
